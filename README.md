# HBuilderX-Light 思源自用主题

本主题借鉴HBuilder软件配色，借鉴UserZYF作者主题css代码,Roy作者主题css js ，使用了链滴id：abbj的ctrl+p窗口改造，使用了langzhou大佬的行内批注。
感谢以上。

目的是打造成一个自己使用没问题的“柔和护眼”主题。

本主题承诺遵守思源默认字体颜色规范。

本主题推荐使用 "Source Han Sans" 开源字体
